Bitcoin Zero-Fee Priority Calculator

What is this?

It's a tool that calculates the Priority value for zero-fee transactions. It'll give you an estimate for how long it will take to have enough Priority to qualify for a free transaction.

What is Transaction Priority?

Transactions generally get included in a block for one of two reasons:

  1. Transaction Fee
  2. Transaction Priority

The transaction fee is the main criteria, but if there is no fee, the transactions needs to have a high enough Priority:

  • Priority is related to the 'age' of the bitcoins you are spending.
  • The older the bitcoins (i.e. number of confirmations), the higher the priority.
  • The larger the number of bitcoins, and the smaller the transaction size, the higher the priority.

As an example, if you are sending a 1 BTC transaction, the previous transaction needs to be roughly 1 day old to have sufficient priority.

How much fee should I pay for my transaction?

Miners typically sort transactions according to Fee Rate (fee per unit transaction size) and include higher fee rate transactions until their maximum block size is reached, or the transaction is below the minimum fee rate. As of Nov 2014 the min fee rate is typically 0.00001000 BTC per kB.

In the future, Bitcoin clients will be able to tell you the optimal fee for getting confirmations, but for now they're mostly hardcoded with respect to the min fee rate.
